# LiteRead

## 对于MVP的几句话：
### 1.对于代码习惯本身就不规范的人来说，mvp不可能是救命稻草可以自己一朝升天，相反使用不规范的编程习惯书写MVP只会让你的代码变得更加难以阅读和维护。
### 2.MVP不是一个开发框架，不要一开始就天天想着去找一个MVP框架来加快开发速度，节省代码书写量。先慢慢理解MVP的设计理念，最后就算自己纯手打也会觉得行云流水。
